# dialloboye.github.io

Personal site — served by GitHub Pages from this repo.

## Publish (one-time)
1. Create a repo named exactly `dialloboye.github.io` on GitHub (public).
2. Push this folder to it (`main` branch).
3. Wait ~1 minute: the site is live at https://dialloboye.github.io
